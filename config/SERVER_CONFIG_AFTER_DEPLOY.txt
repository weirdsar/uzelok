После успешного деплоя с GitHub откройте на сервере (Beget) файл config/config.php
(он не в репозитории) и при необходимости добавьте/обновите:

1) Яндекс Метрика (счётчик 108717789; в layout уже есть дефолт, если ключа нет):
   'yandex_metrika_id' => 108717789,
   Чтобы отключить счётчик: 'yandex_metrika_id' => 0,

2) Заявки с сайта на почту:
   в массиве 'email' ключ 'to':
   'to' => 'ananev-dm@mail.ru',

Сохраните файл в UTF-8. Перезапуск PHP не обязателен для типичного FPM.
